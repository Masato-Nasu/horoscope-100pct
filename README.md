🌟 100% 星占い

100% 星占い は、太陽‐月の伸開角を本格的に計算（Meeus 法）し、毎日の運勢を「100% スコア」で表示する占いアプリです。
12星座の一般論ではなく、恋愛・仕事・金運・健康 を数値（バー表示）と短い解説で具体化します。

🔮 特長

本格的な暦算（Meeus） — 太陽・月の黄経から伸開角を算出

100% スコアバー — 各分野の調子を一目で把握

ジャンル別アドバイス — 恋愛・仕事・金運・健康をコンパクトに解説

任意日も占える — 解析日（今日／指定日）を自由に設定

オフラインでも軽快 — PWA＋キャッシュでサクサク

🧭 Meeus 暦算とは

本アプリの天体計算は、ベルギーの天文学者 Jean Meeus（ジャン・ミュース） による
『Astronomical Algorithms』（1998, Willmann–Bell刊）に基づいています。

Meeus法は、NASA/JPL が用いる DE430/DE440 天体暦 に匹敵する精度を持ち、
太陽・月・惑星の位置を ±0.5° 程度の誤差で算出 します。
ブラウザ内で完結する軽量演算ながら、実際の天文観測値とほぼ一致する信頼性の高い手法です。

Meeus 暦算の特長：

🪐 NASA互換精度：理論式展開により高精度を維持

⚙️ 軽量で高速：ブラウザでもリアルタイム演算可能

📈 長期安定性：紀元前2000年〜西暦3000年まで対応

🌍 完全オフライン動作：外部APIなしで天体演算が可能

💡 スマートフォンの中で、実際の「宇宙時計」が動いているような設計です。

⚙️ アルゴリズム概要（現行実装）

天体位置の計算（Meeus）
解析日の正午（＋出生時刻補正）における 太陽・月の黄経 を算出し、
伸開角 Δ = λ(月) − λ(太陽) を 0–360° で正規化。

相（アスペクト）近傍の評価
良相（60°/120°/240°/300°）に近いほど加点、
0°/180°/270° 近傍は減点（±15〜30°の範囲で評価）。

カテゴリ別スコア化（0–100%）
角度スコアをベースに、カテゴリごとの微調整（恋愛+、金運−など）と
“当日固定”の小さな揺らぎを加えて 100% スケールにマップ。
→ 結果は スコアバー＋数値 で表示。

文章生成（重複ガード）
カテゴリごとのテンプレートから、同一日および直近履歴と被りにくい文言を選出。


📰 ニュース & 季節感

ニュースは GNews API を使用（最大2件の見出し）。
失敗時は NHKニュースRSS に自動フォールバック。

解析日の季節トーン（春夏秋冬の簡潔な一言）を併記。
